# PasswordManager

Десктопное приложение для локального хранения учетных данных и генерации паролей, разработанное на платформе WPF (C#). Приложение обеспечивает базовую защиту данных за счет симметричного шифрования паролей перед их сохранением на диск.

## Функционал

* Хранение учетных записей (Сайт/Приложение, Логин, Пароль) в локальном текстовом файле.
* Автоматическая генерация случайных буквенно-цифровых паролей заданной длины (12 символов).
* Маскирование пароля в поле ввода графического интерфейса.
* Автоматическая дешифрация и отображение сохраненных данных при запуске приложения.

## Технический стек

* **Платформа:** .NET / WPF (Windows Presentation Foundation)
* **Язык программирования:** C#
* **Хранение данных:** Локальный файл `passwords.txt` (структура с текстовым разделителем `;`)
* **Криптография:** Алгоритм AES (Advanced Encryption Standard), пространство имен `System.Security.Cryptography`

## Архитектура и безопасность

Приложение использует симметричный алгоритм шифрования AES для обработки паролей перед их записью в файл. В текущей реализации ключи шифрования (Key и IV) фиксированы внутри исходного кода, что обеспечивает защиту файла данных от прямого чтения в текстовом редакторе, но требует доработки для использования в продакшене.

## Инструкция по запуску

### Требования

* Среда разработки Visual Studio 2022 (или новее) с установленной рабочей нагрузкой ".NET Desktop Development".
* SDK .NET 6.0 или выше.

### Сборка и запуск

1. Клонируйте репозиторий:
```bash
git clone [https://github.com//PasswordManager.git](https://github.com/amadesya/PasswordManagerWPF)

```


2. Откройте файл решения (`.sln`) в Visual Studio.
3. Выберите конфигурацию `Release` или `Debug`.
4. Соберите проект и запустите приложение (клавиша F5).
